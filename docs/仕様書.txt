暗号メーカー 仕様書


============================================================
概要
============================================================

アプリ名:   暗号メーカー
ジャンル:   暗号変換・復号ツール

どんなアプリ?
  テキストをさまざまな暗号方式で暗号化・復号するツール。
  シーザー暗号、モールス信号、Base64、ROT13、アトバシュ暗号、
  ヴィジュネル暗号など多数の暗号方式に対応。
  入力テキストをリアルタイムで変換し、ワンタップでコピー可能。
  暗号の仕組みの解説付きで、学習ツールとしても使える。

誰が使う?
  暗号に興味がある人、友達と秘密のメッセージを交わしたい人、
  情報セキュリティの基礎を学びたい人。


============================================================
機能一覧
============================================================

対応する暗号方式

  古典暗号
    シーザー暗号: 文字をN文字ずらす (シフト数を指定可能)
    ROT13: アルファベットを13文字ずらす (シーザー暗号の特殊ケース)
    アトバシュ暗号: A↔Z, B↔Y のように逆順に置換
    ヴィジュネル暗号: キーワードを使った多表式暗号
    スキュタレー暗号: 棒に巻き付ける転置暗号 (列数を指定)
    レールフェンス暗号: ジグザグに並べ替える転置暗号 (段数を指定)
    ポリュビオスの暗表: 5x5グリッドで文字を座標に変換

  符号・エンコード
    モールス信号: テキストをドットとダッシュに変換 (音声再生対応)
    Base64: テキストをBase64エンコード/デコード
    バイナリ: テキストを2進数表現に変換
    16進数(HEX): テキストを16進数表現に変換
    Unicode: テキストをUnicodeコードポイントに変換

  現代暗号(簡易版)
    XOR暗号: キーとのXOR演算で暗号化
    置換暗号: アルファベットの対応表を自分で設定して暗号化

暗号化 / 復号 切替
  ワンタップで暗号化 ↔ 復号を切り替え
  入力テキストをリアルタイムで変換結果を表示

モールス信号の音声再生
  Web Audio APIで「短音(ドット)」と「長音(ダッシュ)」を再生
  速度調整スライダー
  再生 / 停止ボタン

暗号の解説パネル
  各暗号方式の仕組みをわかりやすく図解
  歴史的背景の簡単な説明
  セキュリティの強度(古典暗号は簡単に解読できるという注意)

お気に入り
  変換結果をお気に入りに保存
  暗号方式 + 入力テキスト + 設定(シフト数等) + 変換結果をセットで保存

テーマ
  ダークモード / ライトモード切り替え


============================================================
画面構成
============================================================

ヘッダー
  アプリ名 + ロゴ (鍵アイコン)
  ナビゲーション (変換 / 一覧 / お気に入り)
  テーマ切替ボタン

変換画面 (メイン)
  暗号方式セレクター (ドロップダウン or グリッドアイコン一覧)
  暗号化 / 復号 切替トグル
  パラメータ設定エリア (暗号方式による)
    シーザー: シフト数スライダー (1〜25)
    ヴィジュネル: キーワード入力
    スキュタレー: 列数入力
    レールフェンス: 段数入力
    XOR: キー入力
  入力テキストエリア
  変換結果テキストエリア (読み取り専用)
  コピーボタン / お気に入り保存ボタン
  解説を見るボタン (パネル展開)
  モールス信号の場合: 再生ボタン + 速度スライダー

暗号一覧画面
  全暗号方式をカード形式で一覧表示
  カテゴリタブ (古典暗号 / 符号・エンコード / 現代暗号)
  各カードにアイコン + 名前 + 一行説明
  タップで変換画面に遷移

解説パネル (スライドイン)
  暗号名 + カテゴリ
  仕組みの図解 (文字の変換過程をビジュアル表示)
  歴史・由来
  セキュリティの強さ目安

お気に入り画面
  保存した変換結果の一覧
  各カードに暗号方式名 + 入力テキスト(一部) + 日時
  タップで詳細表示 + 再コピー

============================================================
暗号変換ロジック設計 (ciphers.js)
============================================================

各暗号は独立した関数として実装
  caesarEncrypt(text, shift) / caesarDecrypt(text, shift)
  rot13(text)
  atbashEncrypt(text) / atbashDecrypt(text)
  vigenereEncrypt(text, key) / vigenereDecrypt(text, key)
  scytaleEncrypt(text, columns) / scytaleDecrypt(text, columns)
  railFenceEncrypt(text, rails) / railFenceDecrypt(text, rails)
  polybiusEncrypt(text) / polybiusDecrypt(text)
  morseEncrypt(text) / morseDecrypt(text)
  base64Encode(text) / base64Decode(text)
  toBinary(text) / fromBinary(text)
  toHex(text) / fromHex(text)
  toUnicode(text) / fromUnicode(text)
  xorEncrypt(text, key) / xorDecrypt(text, key)
  substitutionEncrypt(text, table) / substitutionDecrypt(text, table)

日本語対応
  シーザー暗号: ひらがな・カタカナもシフト対応 (あ→い のように)
  モールス信号: 和文モールスも対応
  その他のアルファベットベースの暗号: 日本語部分はそのまま通す

モールス音声再生
  ドット: 100ms の正弦波 (800Hz)
  ダッシュ: 300ms の正弦波 (800Hz)
  文字間: 300ms の無音
  単語間: 700ms の無音
  速度スライダーで倍率調整 (0.5x〜2.0x)
